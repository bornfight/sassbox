    px-to-vw

    @function px-to-vw($target: 0, $context: 1440) { ... }

    Description

    Function for converting pixel values to viewport width units

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $target

    number of pixels to convert

    		Number0
    $context

    viewport size used for conversion

    		Number1440

    Returns

    String —

    converted viewport units

    Example

    px-to-vw(144)
// 10vw
    px-to-vw(20)
// 1.38889vw

    breakpoint

    @mixin breakpoint($mq01: 0, $mq02: null, $maxmin: max-width, $orientation: null) { ... }

    Description

    Mixin for abstracting media queries

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $mq01

    max width or min width breakpoint, depending on the $maxmin parameter

    		Number0
    $mq02

    max width breakpoint, used if two breakpoints are passed

    		Numbernull
    $maxmin

    determines which type of breakpoint will be used if one is passed (max-width or min-width)

    		Stringmax-width
    $orientation

    used for setting additional orientation in a media query (can be landscape or portrait)

    		Stringnull

    Content

    This mixin allows extra content to be passed (through the @content directive). Role: write SCSS that belongs to the generated media query

    Example

    .body {
    @include breakpoint(960px) {
        background-color: grey;
    }
}
    .body {
    @include breakpoint($breakpoint-big, null, min-width) {
        background-color: grey;
    }
}

    Output

    Wraps content passed inside a media query
